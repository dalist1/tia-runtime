# max compatibility note

`max` is now the legacy compatibility alias for **tia**.

Use the tia docs instead:
- `scripts/TIA.md`

Primary install:

```bash
bash install.sh tia install
```

Primary command:

```bash
tia pi
tia opencode
tia status
```

By default, `install-tia.sh` also creates a `max` alias for older workflows.
