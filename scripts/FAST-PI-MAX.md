# archived fast-pi-max note

`fast-pi-max` is no longer a separate supported mode.

Its startup and tool optimizations have been folded into **tia** so there is only one user-facing runtime mode from this project.

Use:

```bash
bash install.sh tia install
```

Or directly:

```bash
bash scripts/install-tia.sh install
```

If you run `scripts/install-fast-pi-max.sh`, it now delegates to `install-tia.sh`.
